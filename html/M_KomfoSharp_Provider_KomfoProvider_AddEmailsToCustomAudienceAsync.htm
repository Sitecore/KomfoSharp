Parameters

accessToken
Type: SystemString
The access token.
customAudienceId
Type: SystemString
The custom audience identifier.
emails
Type: System.Collections.GenericIEnumerableString
The emails. If hashWithSha256 is set to true the emails have to be normalized by trimming leading and trailing whitespace and converting all characters to lowercase. Maximum 5000 entries could be sent in single call.
hashWithSha256 (Optional)
Type: SystemBoolean
if set to true the emails will be hashed by SHA256 before sending to Komfo.
isHashed (Optional)
Type: SystemBoolean
if set to true the emails are hashed already.

Return Value

Type: TaskInt32
The number of emails requested to be added to the custom audience.
