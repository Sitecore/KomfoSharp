KomfoSharp

KomfoSharp provides API that are used to make calls to Komfo REST API with the simple "Client Credentials Grant" protocol.

You can perform the following actions using KomfoSharp:

  • Authenticate and authorise with the API.
  • Access the stream and user profile metrics for the fans of Twitter channels.
  • Manage Custom Audiences - create Custom Audience and add/remove users to it.
  • Automatically handle the Komfo API rate limits.
  • Automatically handle the access token expiration.
