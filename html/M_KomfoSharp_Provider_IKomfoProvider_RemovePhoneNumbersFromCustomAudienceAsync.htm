IKomfoProviderRemovePhoneNumbersFromCustomAudienceAsync Method
Removes phone numbers from the custom audience asynchronously.

Namespace: KomfoSharp.Provider
Assembly: KomfoSharp (in KomfoSharp.dll) Version: 1.0.0.0 (1.0.0.0)
Syntax
C#
Copy
Task<int> RemovePhoneNumbersFromCustomAudienceAsync(
	string accessToken,
	string customAudienceId,
	IEnumerable<string> phoneNumbers,
	bool hashWithSha256 = false,
	bool isHashed = false
)

Parameters

accessToken
Type: SystemString
The access token.
customAudienceId
Type: SystemString
The custom audience identifier.
phoneNumbers
Type: System.Collections.GenericIEnumerableString
The phone numbers.If hashWithSha256 is set to true the phone numbers have to be normalized by removing any symbols, letters, and any leading zeros. Maximum 5000 entries could be sent in single call.
hashWithSha256 (Optional)
Type: SystemBoolean
if set to true the phone numbers will be hashed by SHA256 before sending to Komfo.
isHashed (Optional)
Type: SystemBoolean
if set to true the phone numbers are hashed already.

Return Value

Type: TaskInt32
The number of phone numbers requested to be removed from the custom audience.
Examples
Copy
var entriesReceived = await komfoProvider.RemovePhoneNumbersFromCustomAudienceAsync("<your_access_token>", customAudienceId, new[] { "15559876543", "15559876544", "15559876545" });
Console.WriteLine("Entries received: {0}", entriesReceived);
